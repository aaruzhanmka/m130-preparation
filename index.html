Квалиметрия

Наука о количественной оценке качества объектов. Главная логика: свойства → показатели → весомости → обобщённая оценка.

Коэффициент весомости

Показывает значимость одного показателя среди других. Часто определяется экспертными методами.

Дифференциальный метод

Сравнивает единичные показатели оцениваемого изделия с базовыми.

Комплексный метод

Даёт одну обобщённую оценку по нескольким свойствам.

Интегральный показатель

Связывает полезный эффект от эксплуатации с затратами на создание и эксплуатацию.

Шкала отношений

Есть абсолютный ноль, можно сравнивать во сколько раз одно значение больше другого.

Испытания

Экспериментальное определение характеристик объекта при воздействии, функционировании или моделировании.

Протокол испытаний

Содержит сведения о продукции, средствах испытаний, условиях, результатах и вывод о соответствии НТД.

Виброустойчивость

Способность выполнять функции во время вибрации.

Вибропрочность

Способность сохранять работоспособность после воздействия вибрации.

Ударная устойчивость

Способность выполнять функции во время ударного воздействия.

Ударная прочность

Способность сохранять параметры после ударного воздействия.
